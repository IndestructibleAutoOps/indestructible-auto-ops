# Browser Screenshots

This directory contains visual feedback screenshots captured during browser automation testing.

## Contents

The screenshots are named using the following convention:
- `visual-feedback-YYYYMMDD-HHMMSS.png`

Where:
- `YYYYMMDD` represents the date (Year, Month, Day)
- `HHMMSS` represents the time (Hour, Minute, Second)
