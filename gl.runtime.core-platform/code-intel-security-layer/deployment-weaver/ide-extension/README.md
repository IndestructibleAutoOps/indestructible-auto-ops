# IDE Extension

## Overview
Provides IDE extensions for VS Code, IntelliJ, and other popular IDEs.

## Features
- Real-time code analysis
- In-editor suggestions and fixes
- Integration with build pipeline
- Customizable rules and patterns

## Supported IDEs
1. **VS Code**: TypeScript/JavaScript extension
2. **IntelliJ IDEA**: Java/Kotlin plugin
3. **PyCharm**: Python plugin
4. **Visual Studio**: C# extension

## Installation
```bash
# VS Code
code --install-extension gl-codeintel-vscode

# IntelliJ
# Install from JetBrains Marketplace
```

## Usage
The extension automatically:
- Analyzes code as you type
- Shows security warnings
- Suggests performance improvements
- Provides quick fixes
