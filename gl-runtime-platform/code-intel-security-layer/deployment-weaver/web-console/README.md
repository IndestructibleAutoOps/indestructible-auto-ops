# Web Console

## Overview
Web-based dashboard and console for Code Intelligence & Security Layer.

## Features
- Interactive code analysis
- Real-time vulnerability scanning
- Performance profiling visualization
- Architecture mapping and diagrams
- Report generation and export

## Deployment
```bash
npm run build:web
npm run deploy:web
```

## Access
- Local: http://localhost:3000
- Production: https://codeintel.gl-platform.io

## Dashboard Components
1. **Overview**: Summary of codebase health
2. **Security**: Vulnerability reports and trends
3. **Performance**: Performance metrics and bottlenecks
4. **Architecture**: Architecture visualizations
5. **Reports**: Generated reports and exports
